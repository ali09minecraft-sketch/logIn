# نظام تسجيل دخول حقيقي (Node.js + Express + SQLite)

نظام تسجيل دخول كامل يعمل فعليًا: إنشاء حساب، تسجيل دخول، جلسات محمية (sessions)، وتشفير حقيقي لكلمات المرور بـ bcrypt. قاعدة البيانات SQLite — ملف واحد، بدون أي خادم قاعدة بيانات منفصل.

## كيف يعمل

- عند إنشاء حساب: كلمة المرور تُشفّر بـ bcrypt ولا تُخزّن أبدًا كنص صريح.
- عند تسجيل الدخول: نقارن كلمة المرور بالنسخة المشفّرة، ونفتح جلسة محفوظة في قاعدة بيانات منفصلة.
- صفحة /dashboard محمية: أي شخص غير مسجّل دخول يُعاد توجيهه تلقائيًا إلى /login.

## النشر على Render

1. Build Command: npm install
2. Start Command: npm start
3. Environment: Node
4. أضف متغيرات البيئة: SESSION_SECRET و NODE_ENV=production

## ملاحظة عن التخزين

في الخطة المجانية لـ Render، قاعدة بيانات SQLite قد تُمحى عند إعادة تشغيل الخادم بعد فترة خمول. مناسب للتجربة، ولاحقًا يمكن الترقية لتخزين دائم أو قاعدة بيانات سحابية.
